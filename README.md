# RSA Algorithm Encryption

Данное приложение реализует алгоритм шифрования RSA. 
Программа позволяет шифровать и расшифровывать текстовые данные, проверяет корректность вводимых значений и демонстрирует работу асимметричного шифрования.

## 🛠 Технологии

*   **Язык:** C#
*   **Фреймворк:** .NET (WPF)
*   **Платформа:** Windows

## 📁 Структура проекта

*   `Models/` — классы, реализующие логику шифрования RSA.
*   `ViewModels/` — классы для связи логики с интерфейсом (MVVM).
*   `Views/` (в данном случае `MainWindow.xaml`) — графический интерфейс пользователя.
*   `Data/` и `Service/` — вспомогательные модули.
